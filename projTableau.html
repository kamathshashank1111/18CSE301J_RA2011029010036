The Below Data is visualised and it provides the information of the database of a Superstore in the USA.



Introduction

The purpose of this project is to analyze and visualize data related to a superstore in the USA. This data could include sales figures, inventory levels, customer demographics, and other relevant information. By creating visualizations of this data, we can gain insights into the store's performance, identify trends and patterns, and make data-driven decisions to improve the store's operations.
The goal of this project is to provide actionable insights that can help the superstore improve its performance and better meet the needs of its customers. By visualizing data in a clear and meaningful way, we can make better-informed decisions and drive positive change.




Horizontal Histogram (Category VS Sales)

Horizontal histograms are useful when you want to compare the relative sizes of different categories or variables. They are particularly effective when you have a large number of categories or when the category names are long, as they allow you to easily display all the categories and their values without the need for vertical scrolling. In a horizontal histogram, the categories are listed on the y-axis, while the values are displayed on the x-axis. The bars run horizontally from left to right, with their length corresponding to the value of the category they represent.


Line Chart (Monthly Sales)

A line chart, also known as a line graph or a time series chart, is a type of chart that displays data as a series of points connected by lines. Line charts are commonly used to show trends over time or to compare the values of different variables. In a line chart, the x-axis represents time or another continuous variable, while the y-axis represents the value of the variable being measured. Each data point is plotted at the intersection of its x and y values, and the points are connected by straight lines. Line charts are useful for showing changes in data over time and for identifying trends or patterns in the data. They can be used to display data from a single variable or to compare data from multiple variables on the same chart.


Packed Bubbles (Manager Wise Sales)

A packed bubble chart, also known as a bubble chart, is a type of chart that displays data as circles, or bubbles, of different sizes. The bubbles are arranged in a hierarchical structure, with larger bubbles representing higher-level categories and smaller bubbles representing lower-level categories. In a packed bubble chart, each bubble represents a data point and is sized according to the value of a particular variable. The bubbles are typically color-coded to represent different categories or groups. Packed bubble charts are useful for showing relationships between data points, and for identifying patterns or trends in the data. They are often used to display data that has multiple dimensions, such as sales data that includes information on product category, region, and time period.


Stacked Bar Chart (Region Wise Orders Placed)

A stacked bar chart is a type of chart that displays multiple data series as bars, with each bar divided into segments that represent different sub-categories. The segments within each bar are stacked on top of each other, so that the height of each bar represents the total value of the data series it represents. Stacked bar charts are useful for showing how the values of different sub-categories contribute to the overall value of each data series. They can be used to compare the values of multiple data series across different sub-categories or to compare the values of a single data series across different sub-categories over time.
