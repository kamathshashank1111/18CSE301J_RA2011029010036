Introduction

In this assignment, We have utilized multiple tools such as Tableau, Python, D3.js, and Gephi to create compelling visualizations for a given dataset. The purpose of this assignment is to demonstrate our ability to use different visualization tools to analyze and present data in a meaningful way.

To start, We had to hunt for a suitable dataset, which we cleaned and prepared for analysis. This process included tasks such as data cleaning, data transformation, and data aggregation to ensure that the data is suitable for visualization.

Next, We used Tableau to create interactive dashboards and charts that help to visualize the data in a way that is easy to understand. Tableau's drag-and-drop interface makes it easy to create visualizations such as scatter plots, line graphs, and bar charts that can be used to explore the relationships between different variables in the dataset.

Additionally, We used D3.js to create custom visualizations that are not available in Tableau. D3.js is a JavaScript library that allows you to create dynamic, interactive visualizations that can be customized to meet your specific needs. For example, you may have used D3.js to create a custom heatmap or a network diagram that highlights the relationships between different entities in the dataset.

Finally, We used Gephi to create a network graph to visualize the relationships between different entities in the dataset. Gephi is a powerful tool for visualizing complex networks and can be used to create interactive visualizations that allow you to explore the relationships between different entities in the data.
